Predicting Vehicle CO2 Emissions

Find Out More

Why Care?

Currently, climate change is a global problem in which everyone's awareness of their own environmental impact is greatly important. The world's transportation sector accounts for 37% of CO2 emissions from end-use sectors, making a significant part of our awareness needing to be on transportation. We created this product so that individuals can easily have a better understanding of their own car's CO2 emissions in order to become a more informed consumer and individual as we all try and reduce our environmental impacts.

Machine Learning Models

Lasso Linear Regression
Lasso regression is a regularization technique which uses shrinkage, when data is shrunk around a central point like the mean. We got a RMSE of 6.665 and a NRMSE of 0.016
Machine Learning Models
hover your mouse on different images to learn more about each machine learning models
Ridge Regression
Ridge regression is a machine learning model that estimates coefficients in scenarios where the data suffers multicollinearily, which means that the independent variables are highly correlated and it has predicted values far away from the actual values. Apart from using L2 regularization, a procedure that prevents excessive fluctuation of the coefficients, ridge regression also protects the model from overfitting. The root mean square error was 7.660, and the normalized RMSE was 0.0179.
Support Vector Regression
Support Vector Regression is a model based on finding the best line to fit the data points within its margins. For the SVR, we implemented 3 hyper parameters, kernel, tolerance, and cost. After refitting the data, we get the RMSE of 19.86 and the NRMSE as 0.047.
XGBoost
XGBoost is a machine learning library that uses the gradient boosting framework for its model. It works by making a weak learning tree then making a second one that predicts the errors of the previous tree and it repeats this process during training to make its final prediction. With this model we got an RMSE of 114.230 and an NRMSE of 0.268
Random Forest
In Random Forest, we’ve got a collection of decision trees (also-known as “Forest”). Forests generally have high number of predictive variables and huge sample size. This is because it captures the variance of several input variables at the same time and enables high number of observations to participate in the prediction. With this we got an RMSE of 0.262 .

Conclusion

After trying various machine learning models and determining their NRMSE(normalized root mean square error), we determined that the lasso linear regression model would work best for our data. We then went on to narrow down potential inputs, keeping what the user would know about their car in mind. You can now input details about your own vehicle above in order to use our model and get an estimate of your vehicle's CO2 emissions. We hope that with this information about your own impact, you will be a more informed consumer when shopping for new vehicles and when using your current one.
